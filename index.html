Projects

SquatCommute

Graduation Project | 2024

SquatCommute was my graduation project, developed to encourage healthier and more sustainable commuting habits. The project involved creating a motion-tracking device that uses a camera to recognize and count squat movements in real time. Built using Python, and tools like OpenCV, Mediapipe, tkinter, and Numpy. For data handling, we used SQLite database. The system accurately identifies squats through posture detection and body movement analysis. Each completed squat rewards users with points that can be redeemed for discounted public transport tickets, linking fitness with sustainable mobility.

RollingBites

RollingBites is a website developed to support and promote local food trucks by providing an online platform for customers to discover and order from them easily. The platform was built using HTML, CSS, JavaScript, and PHP, with a focus on user-friendly navigation and responsive design. It allowed users to browse nearby food trucks, filter by cuisine or price, and view real-time availability.

SadiqEye

SadiqEye is a UI/UX design project aimed at empowering visually impaired users. The app concept focuses on providing real-time assistance by connecting users with helpers through an intuitive and easy-to-navigate interface.
